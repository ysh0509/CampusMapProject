서비스 시작하기 Explore Technology

The Challenge

  • 실시간 혼잡도를 알 수 없는 불투명한 공간 관리
  • 건물 내 복잡한 구조로 인한 길찾기의 어려움
  • 데이터와 현장 피드백 간의 단절

Our Solution

  • Computer Vision: 실시간 인원 및 점유율 분석
  • Unified Graph: 실내외 끊김 없는 통합 내비게이션
  • Edge-to-Cloud: 실시간 데이터 동기화 및 현장 제어

Core Technology

👁️

Vision Intelligence

YOLOv8 모델을 활용하여 카메라 ROI(관심 영역) 내의 객체를 실시간 검출하고, 공간 점유율을 정밀하게 수치화합니다.

🧠

Advanced Pathfinding

A* 알고리즘을 기반으로 거리, 경사도, 계단 유무, 실시간 혼잡도를 가중치로 계산하여 최적의 경로를 산출합니다.

🌐

Unified Spatial Graph

실외 GPS 좌표와 실내 픽셀 좌표를 하나의 통합 그래프로 연결하여 실내외를 아우르는 Seamless한 이동 경험을 제공합니다.

📡

Real-time Sync

Supabase Realtime 엔진을 통해 분석된 공간 데이터가 관리자 대시보드와 현장 임베디드 디바이스에 즉시 반영됩니다.
